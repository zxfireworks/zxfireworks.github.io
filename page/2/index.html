第二章 (流密码)

掌握流密码的基本思想

流密码的基本思想是利用密钥k产生一个密钥流z=z0z1…,并使用如下规则对明文串x=x0x1x2…加密：y=y0y1y2…=Ez0(x0)Ez1(x1)Ez2(x2)…。密钥流由密钥流发生器产生：zi=f(ki,σi),这里σi是加密器中的记忆元件(存储器)在时刻i的状态,f是由密钥k和σi产生的函数

第四章 (公钥密码)

能够根据RSA算法进行加密与解密计算

选两个保密的大素数p和q
计算n=pq，φ(n)=(p-1)(q-1),其中φ(n)是n的欧拉函数值
选一整数e，满足1 < e < φ(n)，且gcd(φ(n),e)=1，这里的(e,n)就是公钥
计算d，满足de ≡ 1 mod(φ(n))，这里的(d,n)就是私钥
加密过程：对于明文m，c ≡ (m^e) mod(n)，c就是m加密后的密文
解密过程：m ≡ (c^d) mod(n)

第五章 (数字签名)

能够使用Elgamal数字签名体制，进行签名与验证计算

选取大素数p，生成元g，p、g公开
随机选取整数x，1 ≤ x ≤ p-2，计算y ≡ g^x mod(p)
公钥为y，私钥为x
签名算法：对于消息m，随机选取整数k，1 ≤ k ≤ p-2，计算r=g^k mod(p)，s=(H(m)-xr)k^(-1) mod(p-1)
m的签名为(r,s)，其中H为Hash函数
验证算法：接收到消息m和签名(r,m)后，验证(y^r)(r^s) ≡ g^H(m) mod(p)
如果成立则签名有效，反之签名无效

第六章 (哈希函数)

理解弱单向函数与强单向函数的定义

单向函数：已知h,求H(x)=h的x在计算上是不可行的，这一性质称为函数的单向性
弱单向函数：已知x,找出y(y≠x)使得H(y)=H(x)在计算上是不可行的
强单向函数：找出任意两个不同的输入x、y,使得H(y)=H(x)在计算上是不可行的
